# Patent Enrichments

## Getting Started
Activate python venv: `source venv/bin/activate`

install python dependencies: `pip3 install -r requirements.txt`

## Project Structure
```
Patent Grants/                          # Your main project folder
├── main.py                             # Main orchestrator script
├── requirements.txt                    # Dependencies file
├── access_converter.py                 # Access database converter
├── install_dependencies.sh             # Installation script
├── .env                                # API keys and constats
├── 
├── classes/                            # Data processing classes
│   ├── __init__.py                     # Empty file (makes it a Python package)
│   ├── data_models.py                  # Data structures
│   ├── patent_xml_parser.py            # XML parser
│   ├── peopledatalabs_enricher.py      # Data enrichment
│   └── data_exporter.py                # Export functionality
│
├── runners/                            # Step-by-step runners
│   ├── __init__.py                     # Empty file
│   ├── integrate_existing_data.py      # Step 0: Access integration
│   ├── extract_patents.py              # Step 1: USPTO extraction
│   ├── enrich.py                       # Step 2: Data enrichment
│   ├── integrate_dynamics.py           # Step 3: CRM integration
│   ├── automate_email.py               # Step 4: Email automation
│   └── monitor_report.py               # Step 5: Reporting
│
├── output/                             # Generated output files
│   ├── enriched_patents.csv            # (Generated)
│   ├── enriched_patents.json           # (Generated)
│   ├── raw_patents.json                # (Generated)
│   └── new_patents_from_xml.json       # (Generated)
│
├── reports/                            # Analytics and reports
│   ├── pipeline_report_YYYYMMDD.json   # (Generated)
│   └── dashboard_YYYYMMDD.png          # (Generated)
│
├── converted_access_data/              # Converted Access data
│   ├── csv/                            # (Generated by access_converter.py)
│   ├── json/                           # (Generated)
│   └── metadata/                       # (Generated)
│
├── templates/                          # Email templates (optional)
│   └── outreach_email.txt              # Custom email template
│
├── patent_system/                      # Your existing Access databases
│   ├── Database.mdb                    # (From your thumb drive)
│   ├── CompactDatabases.accdb          # (From your thumb drive)
│   └── [other .accdb files]            # (From your thumb drive)
│
├── USPC_Download/                      # Your XML patent files
│   ├── ipgb20250701.txt                # (From your thumb drive)
│   ├── ipgb20250708.txt                # (From your thumb drive)
│   └── [other XML files]               # (From your thumb drive)
│
└── Office20/                           # Additional programs (optional)
    └── [files from thumb drive]        # (From your thumb drive)

```
